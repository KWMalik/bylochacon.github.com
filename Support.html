Git is the version control system developed by Linus Torvalds for Linux kernel development. It is used by small development shops and giants like Google, Red Hat, and IBM, and of course many open source projects.  Proprietary software developers will appreciate Git’s enormous scalability. This book is: for all open source developers, by Git experts to turn you into a Git expert, introduces the world of distributed version control, shows how to build a Git development workflow. - What you’ll learn:  to use Git as a programmer or a project leader, become a fluent Git user, use distributed features of Git to the full, acquire the ability to insert Git in the development workflow, migrate programming projects from other SCMs to Git, learn how to extend Git.
