Center for History and New Media Sources

http://chnm.gmu.edu/worldhistorysources/index.html

The USHMM local conference was my first experience with the museum, and within three years, I was a teaching fellow.  This program treats teachers with respect, as professionals.  It focuses on how to teach difficult material, but also provides teachers with the background information they need to be able to teach it.  There is almost limitless help through the fellows, regional coordinators and museum staff.  The website is among the best I’ve ever used, of any type.
